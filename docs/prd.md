# 📱 중국어 단어 틈새 복습 모바일 앱 기획서 - duyiwuer

## 🎯 핵심 컨셉

**"5초의 틈새로 과외 단어 학습 복습"**

- 중국 초등 1학년 교재 기반 (人教版)
- 틈날 때마다 빠르게 학습, 복습
- 부담 없이, 자주, 반복

## 👤 타겟

- 중국어 과외 받는 초중급자
- 한자 잘 모르는 학습자
- 주 1회 과외, 평소 복습 필요

## 📊 데이터

- **총 380자** (1학년 교재)
    - 识字表: 280자 (읽기)
    - 写字表: 100자 (읽기+쓰기)
- HSK 데이터는 보조 (나중에 확장)

## 💫 핵심 기능

### 메인: 플래시카드

```
앞면: 天
↓ (탭)
뒷면: tiān / 하늘

스와이프:
↑ 완벽 (7일 후)
→ 애매 (내일)
↓ 몰라 (1시간 후)
```

### 특징

- **앱 켜자마자 오늘의 단어 바로 학습** (로딩 없음)
- **한 세션 = 10-20자** (1-2분)
- **写字表는 "쓰기" 표시만** (기술구현제약때문에 획순 없음)
